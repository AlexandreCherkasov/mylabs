# Lab1

## How to

1. Create Vagrantfile
2. Start VM
3. In VM
    1. Set Timezone
    2. Install "Development Tools"
    3. Install bc
    4. Get Kernel Source from kernel.org
    5. Extract tarball
    6. Chande directory
    7. Get Old Kernel Config
    8. Make New Kernel
        1. Set Yes for new option in config
        2. Make
        3. Install
        4. Copy New Config and yum.log to Host
